Abstract

Speech-language disorders such as specific language impairment (SLI), also known as developmental language disorder (DLD), are commonly diagnosed through manual analysis of conversational language samples, a time-consuming and subjective process. This project aims to develop NLP models to support early screening using conversational transcripts from CLARIN TalkBank and CHILDES. The task is a binary classification problem that predicts whether a language disorder is present. Baseline models (majority class, logistic regression) will be compared with transformer-based language models. Performance will be evaluated using the F1 score, precision, recall, and ROC-AUC. The planned analysis includes feature interpretation and subgroup comparisons between age groups and types of disorder to assess clinical utility and generalization.

Introduction / Background / Motivation

Within the current field of speech-language pathology, evaluation of child language development often critically includes expert analysis of an ecologically valid conversational language sample. This is obtained by eliciting roughly 50+ consecutive natural utterances by the client, often with the assistance of an audio recording for later transcription.

Speech-Language Pathologists (SLPs), the clinical professionals who primarily make the diagnosis of language disorders of comprehension and expression (receptive and expressive language, respectively), currently use manual methods to code and analyze language samples in order to determine whether enough evidence exists to suggest the presence of a language disorder, primarily in expressive language.

In a field where early detection and treatment often correspond to better prognosis, time is of the essence, esp. with high caseloads and a relatively low supply of SLPs to meet the need. Given the dearth of the tools in production to assist therapists with language sample assessment, this presents a problem that motivates the development of such tools to assist clinicians.

Given the growing power of natural language processing (NLP) and machine learning (ML) technologies, one might naturally ask whether these technologies can contribute to assisting speech and language clinicians in the identification (and thus treatment) of clients with language delays. If the evaluation time can be reduced, clients could then presumably obtain the help they need more quickly.

Approach

What did you do exactly? How did you solve the problem? Why did you think it would be successful? Is anything new in your approach?

We approached this problem by building a text classification pipeline to identify language disorders from conversational transcripts. First, we aggregated individual utterances into session-level documents to better reflect clinically meaningful language use. For the baseline model serving as a majority classifier, we simply predicted the majority class from this data. For the baseline model using logistic regression, we applied TF-IDF vectorization to convert the text into numerical representations, capturing both word frequency and contextual patterns through unigram and bigram features. These representations were used to train the logistic regression model, providing simple and interpretable benchmarks.

In addition to the baseline methods, we implemented a transformer-based language model. This approach leverages contextual embeddings to capture deeper semantic and sequential patterns that are not represented in TF-IDF features.

We chose this combined approach because baseline models such as majority class and logistic regression establish reference points, while transformer-based models have shown good performance in capturing complex language patterns. Our approach is novel in its systematic comparison between traditional methods and transformer-based models applied to conversational data, with attention to clinically relevant linguistic features.

What problems did you anticipate? What problems did you encounter? Did the very first thing you tried work?

We anticipated challenges related to data sparsity and variability in transcript length. One additional concern was tokenization, as previous studies have shown that disordered speech can introduce difficulties for standard tokenization methods. Disfluencies, repetitions, and non-standard language forms may lead to inconsistent token boundaries, which can affect both TF-IDF feature quality and transformer input representations.

Results

How did you measure success? What experiments were used? What were the results, both quantitative and qualitative? Did you succeed? Did you fail? Why?

We evaluated model performance on conversational transcripts from the CLARIN TalkBank / CHILDES dataset for a binary classification task predicting the presence of a language disorder. Success was measured using standard classification metrics, including precision, recall, F1-score, and ROC-AUC, to account for potential class imbalance and provide a comprehensive assessment of model performance.

We conducted experiments comparing a majority class baseline, a logistic regression model using TF-IDF features, and a transformer-based language model. The baseline models were designed to establish interpretable reference points, while the transformer model aimed to capture deeper contextual and sequential patterns in the transcripts. We also evaluated different input representations, including cleaned text and disfluency-tagged transcripts, to assess the impact of preprocessing.

We compared a majority class baseline with a logistic regression model trained on TF-IDF features. The majority classifier achieved an F1-score of 0.78 for the majority class (label 0) but completely failed to identify the minority class (label 1), with precision, recall, and F1-score all equal to 0. This highlights its limitation in handling imbalanced data, as it predicts only the dominant class.

In contrast, the logistic regression model substantially improved performance across both classes. For label 0, it achieved a precision of 0.82, recall of 0.93, and F1-score of 0.87. More importantly, for the minority class (label 1), it achieved a precision of 0.83, recall of 0.64, and F1-score of 0.72. These results indicate that the model was able to capture meaningful linguistic patterns associated with language disorders, although recall for the minority class remains lower, suggesting some missed cases.

Qualitatively, error analysis suggests that misclassifications often occur in sessions with highly variable or disfluent speech, where tokenization and representation may be less reliable. In particular, disfluency-tagged transcripts may introduce noise that affects model performance. Additionally, shorter or less informative sessions appear more difficult to classify accurately.

Overall, the logistic regression model represents an improvement over the baseline and provides a good, interpretable starting point. While the approach was successful in identifying meaningful patterns, the lower recall for the minority class and sensitivity to disordered speech highlight the need for more advanced models, such as transformer-based language models, to better capture complex linguistic structures.
